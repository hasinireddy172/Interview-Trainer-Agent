# 🤖 AI Interview Trainer Agent

### IBM SkillsBuild – AICTE Internship (June 2026)

An AI-powered Interview Trainer Agent developed using **IBM watsonx.ai**, **IBM Granite Foundation Models**, **IBM Cloud Lite**, and **IBM Bob**. The application helps students prepare for job interviews by analyzing their resumes and generating personalized interview questions, preparation tips, and role-specific guidance.

---

# 📌 Problem Statement

**Problem Statement No. 22 – Interview Trainer Agent**

Develop an AI-powered Interview Trainer Agent that prepares users for job interviews by generating personalized interview questions, preparation strategies, and improvement tips based on their resume, experience level, and target job role using IBM watsonx.ai.

---

# 🎯 Objectives

- Help students prepare for interviews.
- Analyze resumes using AI.
- Generate role-specific interview questions.
- Improve confidence through personalized preparation.
- Provide HR, Technical, and Behavioral interview practice.

# 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Configure the `.env` file with your:

- IBM watsonx.ai API Key
- Project ID
- Region URL

Run the application

```bash
python app.py
```

Open the browser

```
http://127.0.0.1:5000
```
